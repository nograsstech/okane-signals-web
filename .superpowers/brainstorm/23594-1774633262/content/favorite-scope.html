What does "favorite" mean?

Should users favorite individual backtest runs or strategy configurations?

A

Favorite specific backtest runs

Each backtest result is a separate favorite. User can have multiple favorites of the same strategy with different parameters or time periods.

Example
❤️ AAPL EMA (60d) — Win Rate: 45.2% — Jan 15, 2026
❤️ AAPL EMA (60d) — Win Rate: 52.1% — Feb 20, 2026
❤️ TSLA Bollinger (1y) — Win Rate: 58.3% — Feb 18, 2026
B

Favorite strategy configurations

User favorites the combination (ticker + strategy + period + interval). Always shows the latest backtest run for that configuration.

Example
❤️ AAPL + EMA + 60d + 1d — Win Rate: 52.1%
❤️ TSLA + Bollinger + 1y + 1d — Win Rate: 58.3%
C

Let me decide in the browser

I want to see both options visualized in the UI mockup and pick the one that looks better.
