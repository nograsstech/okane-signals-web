Where and how do users see their favorite strategies?

C

Both approaches

Favorites section at top for quick access, plus filter toggle on the main table. Maximum visibility and flexibility.

Pros

  • Always see favorites at top
  • Can filter main table too
  • Best of both worlds

Cons

  • More screen space
  • Slightly more complex UI
